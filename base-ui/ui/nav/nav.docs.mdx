## Navigation Component

A nav component for rendering navigation links.

### Component usage

```js
<Nav>
  <a href="https://bit.dev">link 1</a>
  <a href="https://bit.dev">link 2</a>
</Nav>
```

### Using children to pass all links

Modify the html to see it change live:

```js live
<Nav>
  <a href="https://bit.dev">link 1</a>
  <a href="https://bit.dev">link 2</a>
</Nav>
```
