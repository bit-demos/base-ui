import { Card } from './card';

## Card component

An example of a basic card that

- renders children inside it so any html element or component can be rendered inside it
- styling for basic outline of card

### Component usage

Render component with any text, html or other component inside

```js
<Card>Anything can be rendered inside the card</Card>
```

### Using children to render anything inside

Modify the text to see it change live.

```js live
<Card>Anything can be rendered inside the card</Card>
```
