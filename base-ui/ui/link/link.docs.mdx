import { Link } from './link';

A link component that renders a link with some text

### Renders a link

```js live
<Link href="/home">Home</Link>
```

### Renders a link with styling of a button

```js live
<Link href="/details" buttonClass variation="secondary">
  Buy Now
</Link>
```
