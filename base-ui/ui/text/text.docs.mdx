import { Text } from './text';

Text can be rendered in a p element or span element. The default is set to p if no element prop is added

### Renders a text as a p element

```js live
<Text>text as a p tag</Text>
```

### Renders a text as a span element

```js live
<Text element="span">text as a span tag</Text>
```
