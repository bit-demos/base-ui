import { Layout } from './layout';

A layout component that adds a grid for spacing between child elements rendered inside

### Uses Children so anything can be rendered inside

```js live
<Layout>
  <div>Header</div>
  <div>Body</div>
  <div>Footer</div>
</Layout>
```
