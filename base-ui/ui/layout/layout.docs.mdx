## Layout Component

A layout component that adds a grid for spacing between child elements rendered inside

### Component usage

```js
<Layout>
  <div>Header</div>
  <div>Body</div>
  <div>Footer</div>
</Layout>
```

### Uses Children so anything can be rendered inside

Modify the html to see it change live:

```js live
<Layout>
  <div>Header</div>
  <div>Body</div>
  <div>Footer</div>
</Layout>
```
