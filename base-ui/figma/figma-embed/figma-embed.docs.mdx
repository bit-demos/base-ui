## Figma iframe

iframe for embedding figma designs. This can then be used in your component docs.

By embedding our Figma design:

- see if our design matches our component
- see updated changes to design in real time
- great developer experience

### Component usage

```js
<FigmaEmbed src="https://www.figma.com/embed?embed_host=share&url=https%3A%2F%2Fwww.figma.com%2Ffile%2F0EEVmHM5PG37THkzlbanjX%2FColor-Scheme%3Fnode-id%3D0%253A1" />
```
