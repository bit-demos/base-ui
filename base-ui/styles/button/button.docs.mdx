CSS component can be reused across multiple components by importing it and then using it like you would any css class.

## Adds class of Button to button element

```js live
<button className={button}>Hello</button>
```

## Adds class of Button to an link element

```js live
<a href="/" className={button}>
  Hello
</a>
```
